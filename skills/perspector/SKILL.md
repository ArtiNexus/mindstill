# Thought Perspector — 多面透视

## 核心理念

> **观察改变观察对象。但还是要观察。持续观察、持续记录，
> 总会在某个时候发现需要调整的地方。维度可以换。**

五个维度是工具，不是牢笼。好用就接着用，不好用就换。

## 五维透视

| 维度 | 问题 |
|:-----|:-----|
| 客观侧 | 涉及什么物理事实/外部刺激？不以意志为转移的部分。 |
| 主观侧 | 涉及什么精神状态/情绪/价值判断？被心情左右的部分。 |
| 纠缠关系 | 客观↔主观如何互相影响？螺旋还是单向？ |
| 坍缩态 | 这段洞察是事物的哪个面？还缺哪些面才能看到全貌？ |
| 整合介质 | 需要什么介质（物理/符号/意识）才能把这个面和其他面拼起来？ |

## 输入 → 输出

```
输入: ctx.stages.distiller.distilled + ctx.stages.prober.*
输出: {
  objective: "客观侧",
  subjective: "主观侧",
  entanglement: "纠缠关系",
  collapsed_face: "坍缩态",
  integration_medium: "整合介质",
  new_dimension_needed: null or "需要新维度xxx"
}
```

## 提示词

```
你是多面透视者。用五个维度拆解一段洞察。

五个维度是当前的观察工具，不是唯一的。不是真理。

如果某个维度和这段话无关 → 填"不适用"，不要硬凑。
如果五个维度不足以描述这段话 → 标注"需要新维度：xxx"。

维度：
1. 客观侧：什么物理事实/外部刺激？
2. 主观侧：什么精神状态/价值判断？
3. 纠缠关系：客观↔主观如何互影响？
4. 坍缩态：这是事物的哪个面？缺哪些面？
5. 整合介质：需要什么介质拼接碎片？

  "entanglement": "...或'不适用'",
  "collapsed_face": "...或'不适用'",
  "integration_medium": "...或'不适用'",
  "new_dimension_needed": null
}

用户输入:
{{input}}
```
