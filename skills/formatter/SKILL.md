# Thought Formatter — 格式归档

## 三字段结构

```markdown
## 2026-06-17 16:30 — 一句话标题

> 描述：这段话说了什么（蒸馏后的核心洞察）

### 结论及其依据
- 结论：得出的判断
- 依据：基于什么事实/推理

---
```

## 写入策略

### 主文件

完整卡片追加到目标文件末尾。

### 关联文件

在关联文件中追加引用指针：
```markdown
🔗 参见: [[02_方法论与思维链/洞察力#阳光、细胞与情绪]]
   — 客观刺激→生理→情绪的实例
```

## 安全规则

| 规则 | 说明 |
|:-----|:-----|
| 写前备份 | 目标文件存在时，先 copy 到 `.backup/文件名_时间戳.md` |
| 只追加 | 读已有内容，末尾追加，不在中间插入 |
| 验证写入 | 写完后重读，确认内容已在文件中 |
| 空拒绝 | 无实质内容不写入 |

## 文件命名

`{层级前缀}_{关键词}.md`

示例：
- `02_方法论与思维链/洞察力.md`
- `00_认知地基/物质与意识.md`
- `05_当前处理与此刻灵感/system-tool项目.md`
