# Thought Distiller — 蒸馏提纯

## 主人说话的三种结构

| 结构 | 模式 | 蒸馏策略 |
|:-----|:-----|:--------|
| **类比型** | "A 像 B，因为 C" | 保留类比结构："A具备B的结构特质：C" |
| **展开型** | "铺垫 → 事例 → 结论" | 找结论标记词，取标记词后面的句子 |
| **嵌套型** | "说A → 跳B → 跳C → 回A" | 找最后一次回到主线的结论 |

## 结论标记词

```
"大概率就是"、"我的判断是"、"很有可能是"、"我分析"、
"往往就是"、"所以"、"就是说"、"其实"、"你看"
```

找到了 → 标记词后面的句子就是蒸馏目标。
没找到 → 用"就像"/"举个例子"识别类比 → 提取类比关系。
都没有 → 用整段话的最后一句。

## 类比保留规则

> "Agent像蒲公英" → 蒸馏结果不是"Agent是一种工具"
> 而是"Agent具备被驱动的结构能力，但依赖外部触发才能发挥作用"
>
> **保留类比的结构，不把类比压扁成标签。**

## 输入 → 输出

```
输入: ctx.input.raw_text
输出: {
  distilled: "一句话核心洞察",
  structure: "analogy|unfolding|nested",
  layers: [],
  wastage_note: "去掉了什么",
  confidence: 0.85
}
```

## 提示词

```
你是思想蒸馏器。从口语化的长文本中提炼核心洞察。

先判断这段话是哪种结构：
- 类比型：有"像""比如""举个例子" → 保留类比结构
- 展开型：铺垫+事例+结论 → 找结论标记词
- 嵌套型：多话题跳跃 → 找最后一次回到主线的结论

结论标记词："大概率就是""我的判断是""很有可能是""我分析""往往就是""所以""就是说""其实"

蒸馏规则：
1. 类比保留结构，不压扁
2. 去掉语气词/填充词/自我反问
3. 如果有多层洞察，分别提炼

输出 JSON:
{
  "distilled": "一句话核心洞察",
  "structure": "analogy|unfolding|nested",
  "layers": [],
  "wastage_note": "去掉了什么",
  "confidence": 0.85
}

用户输入:
{{input}}
```
