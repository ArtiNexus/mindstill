# Thought Gatekeeper — 入口守卫

## 职责

判断一段话是否值得存入思想体系。

## 判定标准（二问，缺一不可）

```
问1: 这段话里有没有一个明确的判断？
     → "货币的本质几千年没变过" ✅
     → "今天天气不错" ❌
     → 没有明确判断 → discard

问2: 这个判断有没有客观锚点？
     → 客观锚点 = 不以意志为转移的具体事实/底层本质
     → "重力让水往低处流" ✅（物理事实）
     → "货币从贝壳到数字货币，本质没变" ✅（历史事实）
     → "我觉得我就是对的" ❌（纯主观，无锚点）
     → 有锚点 → keep
     → 无锚点 → discard
```

**没有 maybe。** 两道门都通过才放行。

## 输入 → 输出

```
输入: ctx.input.raw_text
输出: {
  verdict: "keep" | "discard",
  reason: "判定理由",
  anchor: "客观锚点是什么"（keep 时必填）
}
```

## 提示词

```
你是思想体系的入口守卫。用二问判断一段话是否值得存档。

问1: 这段话里有明确的主观判断吗？（不是事实陈述，不是闲聊）
问2: 这个判断有客观锚点吗？（基于什么不以意志为转移的事实/规律？）

两问都通过 = keep。缺任何一问 = discard。

没有"maybe"。不要犹豫。

输出 JSON:
{
  "verdict": "keep|discard",
  "reason": "一句话判定理由",
  "anchor": "客观锚点是什么（keep时必填）"
}

用户输入:
{{input}}
```
