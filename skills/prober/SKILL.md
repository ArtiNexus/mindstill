# Thought Prober — 追问穿透

## 核心理念

> **人的思维是螺旋闭环。追问没有终点，只有"当下够用"的解释层。**

三层追问不是通往真理的阶梯，而是扒开壳看看里面是什么——够用了就停。

## 追问链

```
第1问: "所以呢？"
    → 扒开第一层壳。这句话表面在说什么？隐含什么？

第2问: "这说明了什么？"
    → 扒开第二层壳。这个隐含假设意味着什么？

第3问: "这层解释够用了吗？"
    → 够用了 → 标注"解释充分"
    → 不够 → 标注"还有更深层可挖，暂停于此"
    → 不假装"到底了"
```

## 输入 → 输出

```
输入: ctx.stages.distiller.distilled
输出: {
  layer1: "第1问结果",
  layer2: "第2问结果",
  layer3: "第3问结果",
  sufficiency: "sufficient|insufficient",
  note: "如果不够，更深层可能的方向"
}
```

## 降级条件

`distiller.confidence < 0.3` → 跳过 prober，直接用原文进 perspector。蒸馏质量太差，追问没意义。

## 提示词

```
你是追问穿透者。三层追问，不假装到底。

第1问："所以呢？" — 这句话隐含的前提假设是什么？
第2问："说明了什么？" — 这个假设意味着什么？有什么推论？
第3问："这层解释够用了吗？" — 够解释当前问题了吗？

够用了 → 标注"解释充分"
不够用 → 标注"还有更深层可挖"，但不硬挖

不要输出"底层真理""终极本质"这类词。
你只是在扒开两层壳看看里面长什么样。

输出 JSON:
{
  "layer1": "第1问结果",
  "layer2": "第2问结果",
  "layer3": "第3问结果",
  "sufficiency": "sufficient|insufficient",
  "deeper_hint": "如果不够，更深层可能的方向"
}

用户输入:
{{input}}
```
