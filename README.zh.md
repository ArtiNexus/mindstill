# mindstill · 思想蒸馏工具集

> 从你的话语中蒸馏精华。
> 让你的思想长成一棵树。

---

*mindstill* 是一套**开源 AI Skill 管线**，将原始的非结构化语音转化为结构化的、不断生长的个人知识体系。

你说。它听、过滤、蒸馏、追问、透视、关联、归档——让思想之树不断生长，直到有一天，他人能通过这棵树与你对话。

---

## 哲学

我们说出的话，大部分是噪声。但噪声深处藏着洞察——转瞬即逝、片面、但真实。每一个洞察，都是更大真相的一个**坍缩面**。

mindstill 不自称能看到整头大象。它只是收集碎片，一片一片，通过符号介质拼合：**你自己的思想体系，用 Markdown 写成。**

> *碎片 → 蒸馏 → 追问 → 透视 → 关联 → 归档 → 生长*

---

## 管线

```
你的语音/文字
      │
      ▼
┌──────────┐
│ gatekeeper   有判断？有客观锚点？
└─────┬────┘
      ▼
┌──────────┐
│ distiller    去废话，找结论，类比保留结构
└─────┬────┘
      ▼
┌──────────┐
│ prober       "所以呢？"×3，够用了就停
└─────┬────┘
      ▼
┌──────────┐
│ perspector   五维透视：客观/主观/纠缠/坍缩/介质
└─────┬────┘
      ▼
┌──────────┐
│ linker       维度相交检测——不是话题相似，是共享维度
└─────┬────┘
      ▼
┌──────────┐
│ formatter    时间 + 描述 + 结论及其依据
└─────┬────┘
      ▼
你的思想树（Markdown 文件，6层结构）
```

---

## 六层树

| 层 | 存放什么 |
|:---|:--------|
| `00_认知地基` | 关于"我如何认识世界"的前提假设 |
| `01_价值观` | 不可动摇的底层信念 |
| `02_方法论` | 思考工具、分析框架 |
| `03_规则边界` | 灰色地带、弹性标准 |
| `04_阅历感悟` | 故事、教训、经历 |
| `05_此刻灵感` | 闪念、进行中的项目 |

层是你定义的。管线负责填充。

---

## 使用方式

### 配合 pi agent

克隆到 `~/.pi/skills/`，然后说：

> "记一下" — 全管线自动归档
> "我跟你说" — 先过滤，值得存则存

### 配合你自己的 Agent

每个 Skill 是独立的 Markdown 文件，内含提示词模板。可接入任何 LLM。`orchestrator` Skill 定义了编排逻辑。
